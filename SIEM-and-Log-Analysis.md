# SIEM & Log Analysis Notes

## What is SIEM?
SIEM (Security Information and Event Management) is a system used to collect, analyze, and monitor logs from multiple sources in real time.

## Why SIEM is Important
- Detects security threats
- Monitors user activity
- Helps in incident response
- Centralizes logs from different systems


## What are Logs?
Logs are records of events happening in a system.

### Types of Logs:
- System logs
- Application logs
- Security logs
- Network logs

## Analyzed logs 

- Multiple failed login attempts (Brute force)
- Login from unusual location
- Access at unusual times
- Unknown IP addresses
- Suspicious processes
- Phishing Email

## Basic SIEM Concepts

### 1. Log Correlation
Connecting multiple events to detect attacks.

### 2. Alerting
SIEM generates alerts for suspicious activity.
